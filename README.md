# 清迈国际格斗 ✦ Battle Simulator

**Chiang Mai International Fighting** — A 4v4 turn-based battle simulator built on the JJK alternate universe worldbuilding system.

## Play

Open `battle.html` in any modern browser. No dependencies, no build step.

## Features

- **15 original characters** — Zane, Harrison, Charlotte, Lucas, Vincent, Jake, Cici, Feliz, Michelle, Trex, Jeff, LJ, Madame, Woojoo, Chogun
- Full skill implementations including passives, rarity tiers (R / SR / SSR / SSSR)
- Core battle mechanics: 10% dodge, 10% headshot (×1.5), shield damage reduction, trigger accumulation, stamina management
- Enemy AI with role-aware decision making
- Faction passives (后山兄弟 bonds, Charlotte×Zane bond, etc.)
- Special mechanics: Zane absent passive, Chogun skydive, Jeff離場, Charlotte双人转, Lucas GPA accumulation, Michelle betray, Madame fries stacks

## Battle Rules

| Rule | Value |
|------|-------|
| Base HP | 1000 |
| Base ATK | 100 |
| Starting stamina | 5 / max 8 |
| Stamina regen | +2/round |
| Ultimate trigger | 10 |
| Dodge rate | 10% |
| Headshot rate | 10% (×1.5 dmg) |
| Shield reduction | 50% |

## Rarity Tiers

`R < SR < SSR < SSSR`
